# SportsPro - Ultimate Sports Dashboard

A modern, interactive sports dashboard application with amazing UI features and real-time functionality.

## Features

### 🏆 Live Sports Data
- **Live Scores**: Real-time basketball game scores with auto-updating functionality
- **League Standings**: Complete team rankings with win/loss records
- **Player Statistics**: Top player stats including PPG, APG, and RPG
- **Match Schedule**: Upcoming games with dates and times

### 🎨 Amazing UI Features
- **Dark/Light Theme Toggle**: Seamless theme switching with smooth transitions
- **Animated Counters**: Hero section statistics that count up on page load
- **Interactive Cards**: Hover effects and click animations on all cards
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Gradient Backgrounds**: Beautiful gradient effects throughout the interface
- **Live Indicators**: Pulsing animations for live games
- **Smooth Scrolling**: Navigation with smooth scroll to sections
- **Loading Animations**: Professional loading states and transitions

### 🚀 Interactive Features
- **Sport Filtering**: Filter live scores by sport (All, Football, Basketball, Soccer)
- **Navigation**: Sticky header with active state indicators
- **Keyboard Navigation**: Use arrow keys to navigate between sections
- **Auto-Updates**: Live scores update automatically every 5 seconds
- **Social Links**: Ready-to-use social media integration

## Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: CSS Grid, Flexbox, CSS Variables, CSS Animations
- **Icons**: Font Awesome 6.0
- **Responsive**: Mobile-first responsive design
- **Modern Features**: Intersection Observer API, CSS Custom Properties

## Quick Start

1. Clone the repository
2. Open `index.html` in a modern web browser
3. Or serve it with a local HTTP server:
   ```bash
   python3 -m http.server 8000
   # Then visit http://localhost:8000
   ```
